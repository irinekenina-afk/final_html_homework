Git

Вернуться на главную

  • git init - Создать новый репозиторий Git в текущей папке
  • git clone - Склонировать удаленный репозиторий на локальный компьютер
  • git add - Добавить файлы в индекс для следующего коммита
  • git commit - Зафиксировать изменения в репозитории
  • git push - Отправить локальные изменения в удаленный репозиторий
  • git status - Показать состояние рабочего каталога и индекса
